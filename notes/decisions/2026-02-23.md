## Control experiment — proximity effects ruled out

**Context:** Gemini review flagged proximity effects as a limitation needing explicit address.

**Decision:** Run control experiment before updating paper. Four adjacent token pairs with no compound semantic relationship tested at 2.8B. All four returned zero heads above 0.1 threshold. Screen reader returns 101 heads under identical conditions. Proximity is ruled out.

**Control pairs tested:**
- "cold water" — semantically related, not compound
- "very old" — adjective+noun modifier
- "and then" — function words
- "very old" in longer sentence context

**Result:** Results section updated with control experiment subsection.
